# Danh sách lệnh TGIRR CAD LISP

| Nhóm | Lệnh | Chức năng |
|---|---|---|
| Chọn | SB | Chọn đối tượng trong hoặc cắt qua Polyline kín |
| Chọn | SBS | Chọn đối tượng giống mẫu trong biên kín |
| Chọn | SL / SC | Chọn theo Layer / màu hiển thị |
| Văn bản | CT | Căn Text/MText theo trục và điểm chuẩn |
| Văn bản | RT / NT / FT | Xoay, đánh số, tìm và thay thế văn bản |
| Kích thước | CD | Căn chữ Dimension về vị trí chuẩn |
| Kích thước | FD | Sửa tỷ lệ trình bày, không đổi hệ số giá trị đo |
| Kích thước | OD | Xếp đều Dimension theo khoảng cách |
| Layer | LL / UL | Khóa Layer chọn / mở khóa toàn bộ |
| Layer | OL / ULI | Cô lập / khôi phục trạng thái Layer |
| Layer | ML | Gộp Layer nguồn vào Layer đích an toàn |
| Block | CB / RB / XB | Thống kê, đổi tên, phá Block an toàn |
| Dọn bản vẽ | PU / ZA / CL | Purge, Zoom + Regen, Audit + Purge |
| Trợ giúp | TVL / TGIRRABOUT / TGIRRRELOAD | Danh sách lệnh, thông tin, nạp lại Lisp |

Mỗi nút Ribbon có tooltip tiếng Việt chi tiết. Hãy thử trên bản vẽ sao lưu và dùng `UNDO` sau mỗi thao tác trong lần kiểm tra đầu tiên.
