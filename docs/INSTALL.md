# Cai dat

Yeu cau AutoCAD 2023 64-bit va .NET Framework 4.8.

Build bang `scripts/package.ps1`. Sau do copy ca thu muc `.bundle` vao `C:\ProgramData\Autodesk\ApplicationPlugins`. Neu AutoCAD chan plugin, them thu muc bundle vao `TRUSTEDPATHS` va dat `SECURELOAD=1`. Co the test Lisp doc lap bang APPLOAD file `src/lisp/TGIRR_CAD_LISP_Load.lsp`.

Go cai dat bang cach xoa thu muc bundle va khoi dong lai AutoCAD.
